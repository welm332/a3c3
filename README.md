Electron Text Editor
==
# Name
python用のエディター

***a3c3(auto 3 commands 3)***
* Auto 3
    * auto import 
    * auto learning
    * auto typing

* Commands 3
    * easy shortcut commands
    * remote sending commands
    * palette Commands 

# Build
    1:(yarn | npm)  install
    2:(yarn | npx) run install
# Usage
    1:(yarn | npm)  install
    if no build:
        2:(yarn | npx) run start (files)
    else:
        a3c3  (files)
